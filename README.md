#  تطبيق المذكرات اليومية (Diary App)

تطبيق ويب بسيط وسهل الاستخدام لإدارة المذكرات اليومية باستخدام Flask و SQLAlchemy.

##  نظرة عامة

تطبيق المذكرات اليومية هو تطبيق ويب يسمح للمستخدمين بكتابة وحفظ مذكراتهم اليومية بشكل منظم. يتم حفظ كل مذكرة مع تاريخ ووقت الإنشاء تلقائياً، ويمكن للمستخدمين تصفية مذكراتهم حسب اليوم أو الشهر.

##  المميزا

- **إضافة مذكرات جديدة**: اكتب مذكراتك اليومية بسهولة
-  **عرض جميع المذكرات**: شاهد جميع مذكراتك مرتبة حسب التاريخ (الأحدث أولاً)
-  **فلترة المذكرات**: قم بتصفية المذكرات حسب:
  - جميع المذكرات
  - مذكرات اليوم
  - مذكرات الشهر الحالي
- **عرض تفاصيل المذكرة**: عرض مذكرة واحدة بالتفصيل الكامل
-  **حذف المذكرات**: إمكانية حذف المذكرات غير المرغوب فيها
-  **تصميم جميل ومتجاوب**: واجهة مستخدم عصرية تعمل على جميع الأجهزة
-  **حفظ تلقائي للتاريخ والوقت**: يتم حفظ تاريخ ووقت إنشاء كل مذكرة تلقائياً

##  التقنيات المستخدمة

- **Flask 2.3.3**: إطار عمل Python لتطوير تطبيقات الويب
- **SQLAlchemy 3.0.5**: ORM لإدارة قاعدة البيانات
- **SQLite**: قاعدة بيانات خفيفة وسهلة الاستخدام
- **Jinja2**: محرك القوالب لـ Flask
- **HTML/CSS**: للواجهة الأمامية

##  التثبيت والتشغيل

### المتطلبات الأساسية

- Python 3.7 أو أحدث
- pip (مدير حزم Python)

### خطوات التثبيت

1. **استنساخ أو تحميل المشروع**

```bash
# إذا كان المشروع في مجلد محلي
cd project_mansor
```

2. **إنشاء بيئة افتراضية (اختياري لكن موصى به)**

```bash
# على Windows
python -m venv venv
venv\Scripts\activate

# على Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

3. **تثبيت المتطلبات**

```bash
pip install -r requirements.txt
```

4. **تشغيل التطبيق**

```bash
python app.py
```

5. **فتح المتصفح**

افتح المتصفح وانتقل إلى:
```
http://127.0.0.1:5000
```
##  دليل الاستخدام

### الصفحة الرئيسية

عند فتح التطبيق، ستظهر الصفحة الرئيسية التي تعرض:
- جميع المذكرات المحفوظة مرتبة حسب التاريخ (الأحدث أولاً)
- أزرار التصفية (جميع المذكرات، اليوم، هذا الشهر)
- معاينة مختصرة لكل مذكرة (150 حرف)
- أزرار للقراءة الكاملة أو الحذف

### إضافة مذكرة جديدة

1. انقر على زر "إضافة مذكرة جديدة" في القائمة العلوية
2. اكتب محتوى المذكرة في الحقل النصي
3. انقر على "حفظ المذكرة"
4. سيتم حفظ المذكرة مع تاريخ ووقت الإنشاء تلقائياً

### عرض مذكرة بالتفصيل

1. من الصفحة الرئيسية، انقر على "قراءة المزيد" لأي مذكرة
2. ستظهر صفحة تفاصيل المذكرة مع:
   - التاريخ والوقت الكامل
   - المحتوى الكامل للمذكرة
   - خيارات للعودة أو الحذف

### تصفية المذكرات

يمكنك تصفية المذكرات باستخدام الأزرار في قسم "تصفية المذكرات":
- **جميع المذكرات**: عرض كل المذكرات المحفوظة
- **اليوم**: عرض مذكرات اليوم فقط
- **هذا الشهر**: عرض مذكرات الشهر الحالي فقط

### حذف مذكرة

1. من الصفحة الرئيسية أو صفحة التفاصيل
2. انقر على زر "حذف"
3. أكد الحذف في النافذة المنبثقة
4. سيتم حذف المذكرة نهائياً

##  قاعدة البيانات

### هيكل الجدول

يحتوي التطبيق على جدول واحد `diary_entries`:

| العمود | النوع | الوصف |
|--------|------|-------|
| `id` | Integer | المعرف الفريد (Primary Key) |
| `content` | Text | محتوى المذكرة |
| `created_at` | DateTime | تاريخ ووقت الإنشاء |

### إنشاء قاعدة البيانات

يتم إنشاء قاعدة البيانات تلقائياً عند أول تشغيل للتطبيق. ملف قاعدة البيانات `diary.db` سيظهر في المجلد الرئيسي للمشروع.

##  التخصيص

### تغيير مفتاح الأمان

في ملف `app.py`، قم بتغيير `SECRET_KEY`:

```python
app.config['SECRET_KEY'] = 'your-secret-key-here-change-in-production'
```

**ملاحظة**: في بيئة الإنتاج، استخدم مفتاح أمان قوي وعشوائي.

### تغيير قاعدة البيانات

يمكنك تغيير نوع قاعدة البيانات في `app.py`:

```python
# SQLite (افتراضي)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///diary.db'

# PostgreSQL مثال
app.config['SQLALCHEMY_DATABASE_URI'] = 'postgresql://user:password@localhost/diary'

# MySQL مثال
app.config['SQLALCHEMY_DATABASE_URI'] = 'mysql://user:password@localhost/diary'
```

## API Routes

| المسار | الطريقة | الوصف |
|--------|---------|-------|
| `/` | GET | الصفحة الرئيسية - عرض جميع المذكرات |
| `/add` | GET, POST | إضافة مذكرة جديدة |
| `/entry/<id>` | GET | عرض مذكرة واحدة بالتفصيل |
| `/delete/<id>` | POST | حذف مذكرة |
